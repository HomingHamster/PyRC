PyRC - README

ABOUT

PyRC is a free and open source python IRC client that does not require any non standard modules of libraries.

INSTALL

First you need to make sure that python is installed on your computer. See python.org for python installation instructions.
Once you have completed the last step you are free to run the program, which in most cases is as simple as double clicking on the program's icon.

USAGE

To connect to your chatserver use the following command in the entry box at the bottom of the UI:

/server <your_server> [<port_number>]
